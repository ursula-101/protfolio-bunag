Portfolio Highlights

Recent Projects

Route-Route-anay: A Algorithm-Based Map Navigation System

This project develops a map-based navigation system that uses the A* algorithm to optimize commute routes via local jeepney paths, factoring in both jeepney travel and walking distances to and from public utility vehicle routes. It aims to modernize urban mobility by providing commuters with efficient, accessible, and user-friendly route planning that reduces confusion, integrates fare and distance information, and improves overall travel experience in densely populated cities.

WEVO: Smart Scheduling for a Smarter Campus

WEVO is a web-based event venue management system designed for West Visayas State University to solve issues like double bookings and inefficient scheduling. By offering real-time venue availability, automated reservations, and centralized event coordination, it streamlines campus event management while improving communication, reducing scheduling conflicts, and optimizing facility usage through a user-friendly interface.

GabayAni: AI-Powered Agriculture Chatbot Assistant for Every Filipino Farmer

GabayAni is an AI-driven chatbot accessible via SMS or Messenger, designed to support Filipino rice farmers with real-time, localized agricultural advice. Built for low-bandwidth environments, it leverages natural language processing to answer farmer queries in Filipino and Hiligaynon, offering guidance on crop stages, pest management, and best practices, ultimately empowering smallholder farmers with accessible and timely agricultural knowledge.
